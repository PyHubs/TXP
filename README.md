# Text Editor XP
Text Editor XP is a simple, and beautiful `Notepad` **killer**! It uses Tkinter, and has a very _flat UI_. It is really easy to use, and very basic.

## Features
- Basic features such as Open, Save, Save as
- Read simple text
- Simple to use UI.
